3.1 Introduction

This chapter aims to demonstrate competency with supervised learning techniques.

The workflow is captured using Jira under the issue DSS-7:
    • DSS-14 - Identify compatible datasets
    • DSS-16 - Identify predictor and target variables.
    • DSS-17 - Decide on algorithms and scoring metrics.
    • DSS-18 - Tidy and clean the datasets.
    • DSS-19 - Engineer features where appropriate.
    • DSS-20 - Build and train models.
    • DSS-21 - Evaluate and visualise models.
It was decided to target the following techniques:
    • Partial Least-Squares ((GlenArch?))
    • Linear / Quadratic Regression ((ShaunLatham?))
    • Support Vector Machines ((ShaunLatham?), (GlenArch?))
    • Decision Tree Classifiers ((GlenArch?))
    • Logistic Regression ((ShaunLatham?))
    • K-nearest neighbours ((ShaunLatham?), (GlenArch?))
